# Avatar API
Koa based API for vector avatars

## API Request Types

**Avatar request**
```
http://localhost:3000/[format]?[options]
```

#### Request parameters
##### Format _(required)_
* SVG
* PNG

##### Options
* boy
* girl
* man

### Installing

Clone the repo

```
git clone https://github.com/abhishekashyap/avatar-api
```

To install dependencies

```
npm install
```

To start the app use

```
npm start
```
